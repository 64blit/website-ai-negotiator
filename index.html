## Project Understanding

Based on our discussion, you need a web platform that solves the fundamental problem of negotiation deadlock - where buyers and sellers can't find a deal because neither wants to reveal their true limits.

The Core Innovation

  • Sellers set minimum prices privately
  • Buyers set maximum prices privately
  • The system finds deals without revealing either party's limits
  • Optional price revelation for strategic advantage ($1 fee)
  • Can be run manually or with AI agents

Success Metrics

  • Working negotiation rooms with unique URLs
  • Secure state machine that protects price privacy
  • Clean, responsive web UI that builds user trust
  • AI agents that can negotiate effectively (Phase 2)

## State Machine Flow

1. ROOM_CREATED → Seller sets minimum price
2. BUYER_JOINED → Buyer sets maximum price
3. CHECKING_OVERLAP →
   a. If overlap: DEAL_POSSIBLE → Show accept/negotiate options
   b. If no overlap: NO_DEAL → Allow new prices or exit
4. NEGOTIATING → Users adjust prices
5. DEAL_ACCEPTED or DEAL_TERMINATED

Security & Privacy

## Data Protection

  • Encryption: All data encrypted at rest and in transit (HTTPS)
  • Price Privacy: Prices never revealed unless explicitly authorized
  • User-Managed: Password-based encryption controlled by users
  • Audit Trail: Complete logs for security review

## AI Boundaries

  • Each AI agent only sees its own side's data
  • No cross-contamination between buyer/seller contexts
  • Chat history isolated per party
  • No training on user data
